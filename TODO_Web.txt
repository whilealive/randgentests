Evaluierung:
------------
- LuaTeX -> HTML (Konverter)
    - wahrscheinlich keine passenden Tools
    - OS-Abhängigkeit
    - Interaktivität / Lösungskontrolle gegeben?
    - wie publizieren / online stellen?

- HTML direkt aus gleicher Aussagenbibliothek wie LuaLaTeX (ohne LaTeX-Code)
  - wie publizieren / online stellen? (Eigene Webseite, innerhalb Teams, lokales Netzwerk?)
  - Universalität / Portabilität -> sollte möglichst einfach in jedem Schulzimmer einsatzbereit sein...
  - LaTex-Konverter mit Math-Support nötig (MathML?)

- Externe Lösungen
  - Abokosten
  - Bindung an kommerziellen Anbieter, Abhängigkeit
  - Bibliothek doppelt geführt (einmal für LaTeX, einmal Web)
  - Zukunftssicherheit nicht gegeben
  - gibt es Lösungen im Schulbereich (z.B. Sek)?
  - Wie machen es die FHs / Unis?

- Interaktive PDFs
  - Funktionalität Reader-abhängig
  - Machbarkeit?
  - Wahrscheinlich keine automatische Lösungskontrolle

- GUI-Programm (wie Java-Applet)
  - muss auf div. Systemen laufen (Handys, Tablets, Laptops, OS...)
  - Programmieraufwand sehr gross / schwer einzuschätzen
  - Unterhalt gross, Updates usw.
